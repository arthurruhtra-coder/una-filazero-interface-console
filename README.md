# una-filazero-interface-console
🔐 Sistema de Login em C#

Projeto simples de um sistema de login feito em C# para rodar no console.

📌 O que o programa faz
Mostra uma tela inicial
Pede usuário e senha
Verifica se os dados estão corretos
Exibe mensagem de sucesso ou erro;

▶️ Como executar
Abra o projeto no Visual Studio ou VS Code
Execute o programa
Digite o usuário e senha no console;

⚠️ Observação
Este projeto é apenas para aprendizado.
As credenciais estão fixas no código e não são seguras para uso real.
Projeto básico para praticar C# 🚀
