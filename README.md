# una-filazero-interface-console
🔐 Sistema de Login em C#

Projeto simples de um sistema de login feito em C# para rodar no console.

📌 O que o programa faz
Mostra uma tela inicial
Pede usuário e senha
Verifica se os dados estão corretos
Exibe mensagem de sucesso ou erro;
